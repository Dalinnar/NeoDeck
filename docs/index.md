# Plugin Development

Welcome to the NeoDeck plugin development documentation.

NeoDeck plugins can extend:
- buttons
- commands
- monitors
- frontend scripts
- translations
- routes
- backend systems

Plugins are distributed using the `.deck` format and loaded dynamically at runtime.
